# Dual Screen Recorder v0.1.0

## Highlights

- Record one display or two displays in the same session
- Swap left and right screen order before recording
- Capture microphone audio and system audio together
- Pause and resume while recording
- Export to MP4 and extract MP3 audio from the same recording
- Choose the install directory during setup

## Included In This Release

- Windows installer: `Dual Screen Recorder Setup 0.1.0.exe`

## Notes

- Windows only
- FFmpeg is packaged through `ffmpeg-static`
- If you encounter capture issues, include monitor count, Windows version, and whether system audio was enabled when reporting a bug
