# Dual Screen Recorder

[English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja.md)

Electron、React、FFmpeg を使って構築した Windows 向けデスクトップ録画アプリです。

1 台または 2 台のディスプレイを同時に録画し、マイク音声とシステム音声を一緒に収録し、MP4 として書き出し、必要に応じて音声のみを分離して保存できます。

## 主な機能

- 1 画面または 2 画面の同時録画
- 録画前に 2 画面の左右順を入れ替え可能
- マイク音声とシステム音声の同時録音
- 録画の一時停止と再開
- MP4 書き出し
- 同じ録画から MP3 音声のみを書き出し可能
- インストーラーでインストール先フォルダを選択可能
- デフォルトの出力ファイル名形式: `YYYY-MM-DD-random.mp4`

## 想定ユースケース

- 2 画面チュートリアルの録画
- ソフトウェアデモ
- トレーニングや会議の録画
- ナレーション付きの操作説明

## ダウンロード

ビルド済みインストーラーは GitHub Releases で配布します。

利用手順:

1. 最新のインストーラーをダウンロード
2. インストーラーを実行
3. 必要に応じてインストール先を選択
4. `Dual Screen Recorder` を起動
5. 録画する画面を 1 つまたは 2 つ選択
6. 音声オプションを設定
7. 録画開始

## 開発

### 必要環境

- Windows
- Node.js 22+
- npm 11+

### 開発起動

```bash
npm install
npm run dev
```

### ビルド

```bash
npm run build
```

### Windows インストーラー作成

```bash
npm run dist
```

生成物は `release/` に出力されます。

## プロジェクト構成

```text
src/main        Electron メインプロセス
src/preload     Electron とレンダラー間の安全なブリッジ
src/renderer    React UI と録画ロジック
release/        パッケージ出力
```

## 現在の制限

- Windows を主対象としています
- 書き出し時に FFmpeg を使用します
- 長時間録画は実運用前に環境ごとの検証を推奨します
